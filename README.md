
# QuizApp

QuizApp is an Android application developed using Java and Firebase. It allows users to play quizzes, save questions in bookmarks, and receive marks upon completing the test. The app integrates Firebase Realtime Database for storing and retrieving quiz data. It also provides a beautiful user interface for an enhanced user experience.


## Features

-Quiz playing: Users can play quizzes by answering a series of questions.

-Bookmarking: Users can save questions in bookmarks for future reference.

-Mark calculation: The app calculates and displays marks based on the user's performance in the quiz.

-Firebase Realtime Database: The quiz data is stored and retrieved from Firebase Realtime Database.

-Beautiful UI: The app has a visually appealing and user-friendly interface.


## Prerequisites

Before running the TaskApp project, make sure you have the following:

Android Studio (latest version)

Firebase account
